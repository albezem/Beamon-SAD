Discription¶

The Actual Module¶

This is where the core focus of the repository is.

Location

./beamon/

Purpose

Programs code

When user lunches Beamon as a python package using python interpreter, e.g.

python -m beamon

file __main__.py will be executed. On the other hand __main.py will be executed if beamon executable e.g. runbeamon.exe is called using bash
