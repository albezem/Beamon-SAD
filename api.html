beamon.core.assemble_bc(nodes_bc, Dof)[source]¶

Assemble boundary condition information in a vector. nodes_bc should contain [u_x,u_y,u_z,phi_x,phi_y,phi_z] for each node, which can only contain 0 (free) or 1 (locked). The assembled bc vector could be ex. [1,2,5,6] that locks movement in those numbered directions.

Parameters

  • nodes_bc (integer matrix) – boundary condition information from database

  • Dof (integer matrix) – nodes degrees of freedom

Returns

bc vector

Return type

integer 1 x n vector

change_dof(index: int, u_x=None, u_y=None, u_z=None, phi_x=None, phi_y=None, phi_z=None)[source]¶
change the degree of freedom for a given node with specified node index. DOF specification:

1 is for free movement and 0 is for locked movement in a specified direction.

for example (index=300, u_x=1, u_y=1, u_z=1, phi_x=1, phi_y=1, phi_z=0) locks the rotation in global z-axes direction of node 300.

Parameters

  • index – index of the node

  • u_x – movement in x direction

  • u_y – movement in y direction

  • u_z – movement in z direction

  • phi_x – rotation in x axes

  • phi_y – rotation in y axes

  • phi_z – rotation in z axes

Returns

True: if successfully changed/ False: if node not found
